# 更新日志

## 2025-12-26 - 添加欢迎首页和优化滚动条

### 新增功能
1. **欢迎首页 (Welcome Page)**
   - 精美的入口设计，展示应用标题和图标
   - 两个主要功能入口卡片：
     - AI 场景助手：带有功能标签（镜头选择、闪光灯、风格调整）
     - 器材百科：带有功能标签（R50 功能、闪光灯模式）
   - 底部装饰图标区域，展示摄影相关元素
   - 版权信息显示

2. **页面导航优化**
   - 移除底部 TabBar，改为单页面导航模式
   - AI 场景助手和器材百科页面添加返回按钮
   - 使用 `Taro.navigateTo` 和 `Taro.navigateBack` 实现页面跳转

3. **滚动条优化**
   - 添加自定义滚动条样式（H5 环境）
   - 细窄滚动条设计（6px 宽度）
   - 滚动条颜色与主题配色协调
   - 鼠标悬停时显示主色调
   - 小程序环境使用 `scrollbar-hidden` 类隐藏滚动条

### 技术改进
- 重构路由配置，优化页面层级
