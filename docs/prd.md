# R50 光影私教微信小程序需求文档

## 1. 小程序概述

### 1.1 小程序名称
R50 光影私教 (R50 Master)\n
### 1.2 小程序描述
专门为佳能 Canon R50 相机和神牛 Godox TT685II 闪光灯用户设计的参数建议与知识库工具，帮助摄影爱好者快速获取拍摄参数建议和器材知识。\n
## 2. 核心功能模块

### 2.1 功能 1：首页入口

#### 2.1.1 页面布局
- **AI场景助手入口**：占据主要视觉区域，突出「AI参数咨询师」功能
- **器材百科入口**：并排展示，便于用户快速切换
- **历史记录入口**：新增快捷入口，显示最近一次参数记录缩略图
- **底部导航栏**：优化UI设计，添加对应图标标识

### 2.2 功能 2：AI 参数咨询师

#### 2.2.1 输入部分
- **镜头选择**：单选卡片，支持「55mm 定焦」、「18-150mm 变焦」、「100-400mm 长焦」\n- **闪光灯开关**：开关组件，开启时UI元素高亮显示橙色
- **场景描述**：文本输入框支持自定义输入，预设标签选择（如：夜景人像、户外运动、室内静物、户外风景等）\n- **光线环境**：滑块或按钮（清晨、正午、黄金时刻、夜晚）\n- **天气情况**：下拉选择（晴天、阴天、雨天、雪天、多云）\n- **风格偏好**：左右滑动展示的标签页，包含多种拍照风格（日系小清新、胶片复古、高对比黑白、港风、极简、复古、街头摄影、自然主义、戏剧性、纪实摄影、梦幻风格、经典肖像、商业摄影）\n
#### 2.2.2 交互动作
- 底部设置醒目「生成最佳参数」按钮
- 点击后弹出全屏Loading页面，显示相机过场动画和「正在分析光影...」提示
- 使用mock data展示参数结果

#### 2.2.3 结果展示
- **参数仪表盘**：数字滚动动画显示ISO、光圈(f)、快门速度、白平衡、锐度、反差、饱和度、色调
- **闪光灯卡片**：开启闪光灯时显示模式(TTL/M)、功率(1/64等)、灯头角度
- **操作建议**：简短文字提示（如「请使用M档，开启眼部对焦」）\n
### 2.3 功能 3：历史参数记录

#### 2.3.1 记录展示
- 时间轴排序，显示参数生成时间
- 参数缩略图预览，包含关键参数值
- 支持快速查看完整参数详情

#### 2.3.2 交互功能
- **查看详情**：点击记录进入详细页面，完整展示输入参数和建议参数
- **重新生成**：支持基于历史记录重新生成参数
- **删除记录**：单个删除或批量删除功能
- **收藏标记**：支持标记常用参数配置

#### 2.3.3 UI设计
- 卡片式布局，圆角设计增强亲和力
- 参数卡片采用渐变背景，重要数值使用等宽字体突出显示
- 添加滑动删除手势和心形收藏图标

### 2.4 功能 4：器材百科 (Wiki 页)\n
#### 2.4.1 Canon R50 解释
- A+场景智能
- Tv快门优先
- Av光圈优先
- Servo伺服对焦

#### 2.4.2 Godox TT685II 解释
- TTL自动测光
- M手动模式
- HSS高速同步
- 前/后帘同步

#### 2.4.3 UI风格
- 手风琴组件
- 卡片式布局
- 对应小图标

## 3. 技术规范

### 3.1 技术栈
- **框架**：Next.js 14 (App Router)\n- **样式**：Tailwind CSS
- **组件库**：Shadcn/UI (基于Radix UI)\n- **动画**：Framer Motion
- **图标**：Lucide React
- **状态管理**：Zustand

### 3.2 设计规范
- **背景**：#0f0f0f（深色哑光黑，体现专业感）\n- **主色调（Action）**：#D32F2F（类似佳能红，用于主按钮和核心强调）\n- **辅助色（Flash）**：#F57C00（类似神牛橙，专门用于闪光灯相关元素）\n- **文字**：白色（主要内容）、Gray-400（次要信息）\n- **字体**：Inter或系统默认无衬线字体，参数数字使用等宽字体（Monospace）模拟相机屏幕
- **布局**：移动端优先，使用底部固定导航栏（Bottom Navigation）切换功能

## 4. 交互规范

### 4.1 导航设计
- 底部固定导航栏切换功能模块，添加功能图标标识
- 优化右侧滚动效果，确保视觉流畅性
- 历史记录页面采用列表滑动交互，支持惯性滚动
- 风格偏好标签支持左右滑动切换

### 4.2 交互反馈
- 按钮点击反馈
- 表单输入验证
- 加载状态提示
- 滑动删除时的阻力反馈和完成动画
- 心形收藏的缩放和颜色变化动画
- 左右滑动时的摩擦力反馈和边界阻尼

## 5. 数据规范

### 5.1 数据来源
- Mock data模拟参数分析结果
- 预设器材知识库内容
- 历史记录数据库存储

### 5.2 数据结构
- 参数建议标准化输出格式
- 器材百科分类组织结构
- 历史记录包含输入参数、建议参数、生成时间和操作状态

## 6. 动画规范

### 6.1 Loading动画
- 相机快门开合动画
- 胶片卷动过渡效果
- 镜头光圈收缩扩张
- 快门声音效配合视觉节奏

### 6.2 交互动画
- 按钮按下时的缩放反馈
- 页面切换的滑动过渡
- 参数数字的滚动入场动画
- 卡片滑动时的阴影变化
- 收藏图标点击时的心形扩散效果
- 左右滑动时的平滑过渡动画
