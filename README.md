# R50 光影私教 (R50 Master)

专门为佳能 Canon R50 相机和神牛 Godox TT685II 闪光灯用户设计的参数建议与知识库工具，帮助摄影爱好者快速获取拍摄参数建议和器材知识。

## 功能特性

### 1. 欢迎首页
- **精美入口设计**：两个主要功能入口卡片
- **AI 场景助手入口**：快速进入参数咨询功能
- **器材百科入口**：快速查看器材知识
- **历史记录快捷入口**：右上角快速访问历史记录
- **底部装饰图标**：展示摄影相关图标元素

### 2. AI 场景助手
- **镜头选择**：支持 55mm 定焦、18-150mm 变焦、100-400mm 长焦
- **闪光灯控制**：神牛 TT685II 开关及参数建议
- **场景描述**：预设场景（夜景人像、户外运动、室内静物）+ 自定义场景
- **光线环境**：清晨、正午、黄金时刻、夜晚
- **风格偏好**：日系小清新、胶片复古、高对比黑白
- **精美加载动画**：相机图标动画 + 装饰性图标跳动效果
- **智能参数生成**：自动生成 ISO、光圈、快门速度、白平衡等完整参数建议

### 3. 参数结果页
- **入场动画**：参数生成完成后的过渡动画
- **参数仪表盘**：清晰展示所有相机参数
- **闪光灯参数卡片**：独立展示闪光灯相关设置（开启时）
- **操作建议**：针对性的拍摄建议
- **快捷操作**：再次分析、查看历史记录按钮
- **自动保存**：参数自动保存到历史记录

### 4. 历史记录页
- **时间轴展示**：按时间倒序显示所有历史记录
- **智能时间格式**：刚刚、X分钟前、X小时前、具体日期
- **完整信息**：显示输入参数和生成结果
- **标签展示**：镜头、闪光灯、场景、光线、风格标签
- **参数预览**：快速查看主要参数
- **删除功能**：单条删除或清空所有记录
- **空状态提示**：无记录时的友好提示

### 5. 器材百科
- **Canon R50 知识库**：A+ 场景智能、Tv 快门优先、Av 光圈优先、Servo 伺服对焦
- **Godox TT685II 知识库**：TTL 自动测光、M 手动模式、HSS 高速同步、前/后帘同步
- **手风琴交互**：点击展开查看详细说明

## 设计特色

- **深色专业风格**：#0f0f0f 深色哑光黑背景，体现摄影器材专业感
- **佳能红主色**：#D32F2F 用于主按钮和核心强调
- **神牛橙辅助色**：#F57C00 专门用于闪光灯相关元素
- **等宽字体**：参数数字使用等宽字体，模拟相机屏幕效果
- **渐变与阴影**：精心设计的渐变背景和阴影效果
- **优化滚动条**：自定义细窄滚动条，H5 环境下更美观
- **丝滑动画**：
  - 加载动画：相机图标脉冲 + 波纹扩散效果
  - 入场动画：淡入 + 上移效果
  - 跳动动画：装饰图标的节奏跳动
  - 过渡动画：页面切换的流畅过渡

## 技术栈

- **框架**：Taro + React + TypeScript
- **样式**：Tailwind CSS
- **状态管理**：Zustand
- **图标**：Material Design Icons (MDI)

```bash
pnpm install # 安装依赖
```

```bash
pnpm run lint  # 代码检查
```

## 开发说明

本项目使用 Mock 数据模拟参数生成逻辑，无需后端服务。所有参数建议基于预设算法，根据用户选择的镜头、场景、光线和风格自动计算。历史记录保存在内存中（Zustand store），刷新后会清空。

## 交互流程

1. **欢迎页** → 选择功能入口
2. **AI 场景助手** → 输入拍摄参数 → 点击生成
3. **加载动画** → 1.5秒相机动画展示
4. **参数结果页** → 查看完整参数 → 自动保存到历史
5. **历史记录页** → 查看/删除历史记录
